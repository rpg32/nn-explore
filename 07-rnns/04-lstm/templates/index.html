LSTM — Long Short-Term Memory

A cell state acts as a gradient highway — information flows through time without vanishing.

1 The Cell State Is a Gradient Highway

The Problem with Basic RNNs

In a basic RNN, the hidden state passes through a matrix multiplication and tanh at every step. Gradients must flow backward through all of these — each one shrinks the gradient a little. After 20-30 steps, the gradient is effectively zero. The network can't learn from early characters.

LSTM's solution is beautifully simple: add a second state vector — the cell state — that runs through ALL time steps with only element-wise multiply and add. No matrix multiplication on this path. Gradients flow through it almost unchanged. It's a highway for gradient flow.

The Conveyor Belt Analogy

Think of the cell state as a conveyor belt running through a factory. Items (information) ride the belt from one end to the other. At each station, workers can:

The key insight: items on the belt don't get transformed — they're only multiplied (forget) or added to (input). This means gradients flowing backward along the belt don't get squished by repeated matrix multiplications. That's the gradient highway.

Forget gate f = σ( W_f · [h_prev, x] + b_f )
Input gate i = σ( W_i · [h_prev, x] + b_i )
Candidate c̃ = tanh( W_c · [h_prev, x] + b_c )
Output gate o = σ( W_o · [h_prev, x] + b_o )
c_new = f ⊙ c_prev + i ⊙ c̃   ← element-wise only! No matrix multiply on the cell state path
Hidden out h_new = o ⊙ tanh( c_new )

3 RNN vs LSTM — Gradient Flow Compared

Gradient Strength Flowing Backward Through Time

We start with a gradient of 1.0 at the last time step and flow it backward. In the RNN, each step multiplies by W_h and tanh' — the gradient decays exponentially. In the LSTM, the gradient flows through the cell state path, only multiplied by the forget gate (~0.7-0.9) — it stays much stronger over long sequences.

Things to Try

  • Step through "hello" and watch the forget gate. It starts near 1 (all green) because of the bias trick — the LSTM defaults to remembering everything. As characters arrive, some slots start getting erased.
  • Try "aaaaaa" — same character repeated. Watch how the gates settle into a pattern. The cell state should stabilize since the same input keeps arriving. The forget gate learns what's worth keeping.
  • Try "abcabcabc" and watch the cell state evolve. When the pattern repeats, the cell state should show periodic behavior — the LSTM recognizes the cycle.
  • Compare the gradient chart. The RNN gradient (red) drops to near zero within ~15 steps. The LSTM gradient (blue) stays meaningful across 30+ steps. This is why LSTMs can learn long-range dependencies.
  • Look at the candidate vs input gate. The candidate (tanh) shows WHAT the LSTM wants to write. The input gate (sigmoid) controls HOW MUCH actually gets written. Together they're like a careful editor.
  • Watch the hidden state vs cell state. The cell state is the full memory. The hidden state is a filtered view through the output gate. Not everything in memory needs to be output at every step.
