2 The Bag-of-Words Problem

When an MLP processes a sentence, it typically sums or averages the word vectors. This is called a "bag of words" — like dumping the words into a bag and shaking them up.

Since addition is commutative (a + b = b + a), rearranging the words gives the exact same sum. The MLP literally cannot tell the difference.

4 What We Need: Processing With Memory

The fundamental problem: an MLP processes all inputs simultaneously. It has no concept of "first", "then", "after". To handle sequences, we need a network that:

① Processes one element at a time

Instead of seeing all words at once, read them left to right — just like a human.

② Maintains a memory (hidden state)

After reading each word, update an internal state that summarizes what's been seen so far.

③ Lets context influence interpretation

"not" before "good" changes the meaning. The memory of "not" must affect how "good" is processed.

This is exactly what a Recurrent Neural Network (RNN) does. Here's the idea:

Step 1
"dog"
memory: dog
→
Step 2
"bites"
memory: dog bites
→
Step 3
"man"
memory: dog bites man

vs. a different sequence building different memory:

Step 1
"man"
memory: man
→
Step 2
"bites"
memory: man bites
→
Step 3
"dog"
memory: man bites dog

Now the order matters — the hidden state at step 3 is different because the sequence of updates was different. That's the key insight behind RNNs, and we'll build one in the next module.

Things to Try

  • Type "cat chases mouse" and "mouse chases cat" in the comparison box — same words, same MLP output, but very different stories for the cat and the mouse.
  • Try "not bad" and "bad not" — the MLP can't distinguish negation from nonsense. In real language, word order defines grammar and meaning.
  • Add duplicate words — "dog dog bites man" vs "dog bites man man". Now the word SETS differ, so the MLP CAN tell them apart. It only fails when the same words are rearranged.
  • Look at the time series — Rising and Falling have identical sums and means. Any model that aggregates values without considering position will confuse them.
  • Think about music — the notes C, E, G played in order make a major chord. Played as G, E, C they're an inversion. Same notes, different feel. Order is the music.
  • Consider "the dog" vs "dog the" — even two words can carry order-dependent meaning. Articles before nouns vs after make English vs nonsense.
