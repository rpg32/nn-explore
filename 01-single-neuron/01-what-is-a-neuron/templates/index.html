What is a Neuron?

The simplest building block of every neural network

A neuron is the fundamental unit of every neural network. It does something remarkably simple: it takes a few numbers as inputs, multiplies each one by a weight (how important that input is), adds everything up, adds a bias, and outputs a single number.

output = (w₁ × x₁) + (w₂ × x₂) + (w₃ × x₃) + bias

That's it. Every neural network — from simple classifiers to GPT — is made of millions of these. The magic is in how they're connected and how the weights are set. Drag the sliders below to see it in action.

💡 Things to Try

  • Set all weights to 0. What happens to the output? Only the bias remains — weights control how much each input matters.
  • Make w₁ very large (2.0) and the others small. Now x₁ dominates the output — it's the most "important" input.
  • Set all inputs to 0 and adjust the bias. The bias shifts the output even when there's no input signal. It's like a "default" value.
  • Make a weight negative. Now that input decreases the output. Negative weight = inhibition.
  • Try to make the output exactly 1.00. Can you find a combination that hits it?
