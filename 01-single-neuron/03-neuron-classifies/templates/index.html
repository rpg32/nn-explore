Module 01 · The Single Neuron

A Neuron Classifies

Using a single neuron to draw a decision boundary

A single neuron can classify — look at a 2D point and decide which of two groups it belongs to. It computes the weighted sum, passes it through sigmoid, and says: blue (Class A, ≥ 0.5) or orange (Class B, < 0.5).

class = sigmoid( w₁ · x₁ + w₂ · x₂ + bias )    ≥ 0.5 ? A : B

The decision boundary is the straight line where the neuron is exactly 50/50. Adjust the weights to rotate the line, and the bias to shift it. Click the canvas to add your own points, or load a sample dataset.

Things to Try

  • Load "Simple" and adjust w1, w2, bias to get 100% accuracy. The line separating blue from orange is the decision boundary.
  • Load "Diagonal" — you need a different slope. Which weight controls rotation?
  • Load "Overlapping" — the clusters overlap. Can you get above 80%? Some errors are unavoidable.
  • Load "Circle" — blue dots surround orange dots. No straight line can separate them! This is the fundamental limit of a single neuron. (Module 04 fixes this with more layers.)
  • Click to add your own points. Draw a pattern and see if you can separate it.
  • Set w1=0. The boundary becomes horizontal — only x2 matters. Set w2=0 and it goes vertical.
