Loss Functions

Measuring "how wrong" a prediction is — the starting point of learning

Before a network can learn, it needs a way to measure its mistakes. A loss function takes the network's prediction and the true answer, and returns a number: 0 = perfect, bigger = worse.

MSE = ( ytrue - ypred )2      |      BCE = -[ ytrue · log(ypred) + (1 - ytrue) · log(1 - ypred) ]

MSE (Mean Squared Error) treats all errors proportionally. BCE (Binary Cross-Entropy) screams at confident wrong predictions — making it far better for classification. Compare them below.

Things to Try

  • Set truth=1, drag prediction from 0.01 to 0.99. Watch both curves. BCE explodes near 0 — it hates confident wrong answers. MSE stays relatively calm.
  • Set truth=0 and repeat. Now the curves flip — loss is high when prediction is near 1.
  • Look at the curve shapes. MSE is a gentle parabola. BCE is a steep cliff on the wrong side. Which would push the network harder to fix its mistakes?
  • In the batch section, make one prediction very wrong (drag it to the opposite extreme). Switch between BCE and MSE — see how much more BCE penalizes that outlier.
  • Try to minimize the batch mean loss by getting all predictions close to their true values.
