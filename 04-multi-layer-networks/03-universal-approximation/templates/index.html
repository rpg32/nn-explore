Module 04 · Multi-Layer Networks

Universal Approximation

More neurons = finer fit — like refining a mesh in FEM

The Theorem

A network with just one hidden layer and enough neurons can approximate any continuous function to any desired accuracy.

Each ReLU neuron creates a hinge (a kink in the output). More neurons = more hinges = the network can trace finer details. This is exactly like FEM: more elements = finer mesh = better approximation of the true solution.

Each colored line is one hidden neuron's output, weighted by its connection to the output. They sum together to form the blue fit curve above. Each is a "hinge" — flat on one side, sloped on the other.

Things to Try

  • Start with 1 neuron on Sine. One hinge can't curve — terrible fit. Go to 2: slightly better. 4: starting to trace the wave. 16: close. 64: nearly perfect.
  • Try Square Wave. Sharp discontinuities are hard. Even 64 neurons show some ringing near the jumps (like Gibbs phenomenon in Fourier series).
  • Look at the basis functions. Each neuron is a simple hinge. Together they approximate any shape. This is the same as linear FEM elements — each hat function is simple, but enough of them can approximate any solution.
  • Compare Sine vs Step. Smooth functions need fewer neurons for a good fit. Discontinuous functions need many more — just like FEM needs mesh refinement near singularities.
  • Notice: these are untrained random weights! No — wait, they ARE trained. The network learned where to place each hinge and how to weight it. That's the magic: backprop finds the optimal "mesh" automatically.
