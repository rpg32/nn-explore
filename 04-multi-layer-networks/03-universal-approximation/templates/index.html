Universal Approximation

With enough neurons, a single hidden layer can approximate any function

The Universal Approximation Theorem

A feedforward network with one hidden layer and enough neurons can approximate any continuous function to any desired accuracy. This is like mesh refinement in FEM — a coarse mesh (few neurons) gives a rough shape, a fine mesh (many neurons) captures every detail.

Each hidden neuron is a "hinge" — a ReLU that bends at one point. More hinges mean finer control, just like more elements in a finite element mesh give better accuracy.

Things to Try

  • Start with Sine Wave. Watch how 1 neuron gives a crude straight-line approximation, 2 neurons add a bend, 4 start to curve, and 16+ almost perfectly trace the sine wave.
  • Try Square Wave. Even 64 neurons struggle with the sharp edges — this is the Gibbs phenomenon. Smooth activation functions can't make truly sharp jumps.
  • Look at the Basis Functions. Each colored line is one neuron's contribution. The network sums these to build the total curve. Like how FEM elements combine to form the solution.
  • Compare Step Function with 1 vs. 32 neurons. With 1 neuron, the network can only make a single ramp. With 32, it can make a much sharper transition. But it never becomes truly discontinuous.
  • Multi-Frequency function needs many neurons. The function has three frequency components mixed together. You need at least 8-16 neurons to capture all the wiggles.
  • Watch the MSE drop as you add neurons. It's not always monotonic (randomness in training), but the trend is clear: more neurons = lower error.
  • Think about FEM: A 2-element beam gives average stress. A 100-element beam shows stress concentrations. Neurons are like elements — more resolution requires more of them.
