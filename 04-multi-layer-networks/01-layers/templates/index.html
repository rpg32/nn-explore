Module 04 · Multi-Layer Networks

Layers

Organizing neurons into input, hidden, and output layers

Three Types of Layers

Input Layer: Not neurons — just the raw data values passed in. If your data has 2 features (x1, x2), the input layer has 2 nodes. No weights, no activation, no computation.

Hidden Layers: Layers of real neurons between input and output. Each neuron connects to ALL values from the previous layer (fully connected). Each computes: activation(w1×in1 + w2×in2 + ... + bias). This is where the network learns features.

Output Layer: The final layer producing the prediction. For binary classification: 1 neuron with sigmoid. For 10 classes: 10 neurons with softmax.

Why "Fully Connected"?

Every neuron in one layer connects to every value in the previous layer. In the diagram above, each line is a weight. A layer with 4 neurons receiving from 2 inputs has 4×2 = 8 weights plus 4 biases = 12 parameters.

This is why parameter counts grow fast: a 784-input layer (like a 28×28 image) feeding into 256 hidden neurons = 200,960 parameters in just one layer. CNNs (Module 06) avoid this by using local connections.

Things to Try

  • Start with "2 → 1" (no hidden layers). Just input directly to output — a single neuron. This is Module 01. Only 3 parameters (2 weights + 1 bias).
  • Switch to "2 → 4 → 1". Now there's a hidden layer with 4 neurons. Watch the parameter count jump to 17. Each hidden neuron receives BOTH inputs and computes its own weighted sum.
  • Go to "2 → 8 → 4 → 1". Two hidden layers. 53 parameters. Data flows through 3 stages of transformation.
  • Click on any neuron in the diagram to see its exact computation — which inputs it receives, what weights it uses, the weighted sum, bias, activation.
  • Drag the input sliders. Watch ALL the neuron values change — the input ripples through the entire network. This is the forward pass (Module 04.2).
  • Notice the colors: brighter = larger activation. The network transforms the input through each layer into something different.
