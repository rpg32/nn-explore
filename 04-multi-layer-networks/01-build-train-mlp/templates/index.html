Things to Try

  • Linear dataset, 0 hidden layers (just click - Layer until gone). A single neuron handles this. Now try XOR — it fails completely.
  • XOR dataset, 1 hidden layer with 4 neurons. Train 100 steps. Watch the nonlinear boundary appear. A single neuron couldn't do this!
  • Circle dataset, 1 hidden layer. Add neurons (4 → 8 → 16). More neurons = finer boundary. This is universal approximation — like refining an FEM mesh.
  • Spiral dataset — the hard one. Start with [8] hidden. Doesn't work. Try [16, 8]. Better. Try [32, 16, 8]. Now it traces the spiral. Depth + width matter.
  • Compare activations on Spiral. Build [16, 8], train 200 steps with ReLU vs Sigmoid vs Tanh. ReLU typically converges fastest (Module 03.3 explained why).
  • Watch the decision boundary evolve. On any dataset, train 100 at a time and see the boundary sharpen with each batch.
